any,unknown,never egula holo typescript er kichu type . jegulor maddhome value er type define kora jay.

any: type any er khetre jekono dhoroner value newa jay . jemon : ami jodi type any use kori porobortite chaile ami sheta string dite parbo , number dite parbo , boolean dite parbo etc. ar normally kon dhoroner type ta define na korle sheta automatically any type er moddhe jabe.

unknown: type unknown holo jokhon amra jani na je amader type ta kon dhoroner ashbe. amra ta runtime e bujhte parbo  

never: type never use hoy jokhon amra jani je eita jibone kono din kono kichu return korbe na.
